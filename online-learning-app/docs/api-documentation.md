# API Documentation

## API Endpoints Description

### Authentication

- POST /api/auth/register
- POST /api/auth/login

### Courses

- GET /api/courses
- POST /api/courses
- PUT /api/courses/:id
- DELETE /api/courses/:id

### Enrollments

- POST /api/enrollments
- GET /api/enrollments/user/:userId

### GPT Integration

- POST /api/gpt/chat
