# Online Learning App

## Project Overview

A full-stack online learning platform with:

- **Backend**: Node.js + Express + MongoDB
- **Mobile App**: React Native
- **Features**:
  - User authentication (Student/Instructor roles)
  - Course management
  - Enrollment system
  - GPT-3 powered educational chat assistant

See `docs/setup.md` for detailed setup instructions.

- Node.js & Express
